# POP VAULT

Tienda premium de figuras Funko Pop hecha en Flutter.

UI dark glassmorphism: fondo aurora animado, paneles de vidrio esmerilado
(backdrop blur), acentos neon con glow, formas squircle y microinteracciones
de resorte.

## Ejecutar

```bash
flutter pub get
flutter run -d edge   # o el dispositivo que prefieras
```
