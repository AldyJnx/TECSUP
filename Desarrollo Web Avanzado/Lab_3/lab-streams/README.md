# Lab Streams - Node.js

## Estructura del Proyecto

```
lab-streams/
├── index.js
├── actividad1.js
├── actividad2.js
├── actividad3.js
├── actividad4.js
├── tarea1.js
├── tarea2.js
├── datos.txt
├── entrada.txt
├── texto.txt
└── package.json
```

## Ejecución

### Menu Principal
```
node index.js
```

## Ejecución Individual

### Actividad 1: Lectura de archivo con Streams
```
node actividad1.js
```

### Actividad 2: Escritura con Streams
```
node actividad2.js
```

### Actividad 3: Compresión con Pipes
```
node actividad3.js
```

### Actividad 4: Manejo de errores y Backpressure
```
node actividad4.js
```

### Tarea 1: Convertir texto a MAYÚSCULAS
```
node tarea1.js
```

### Tarea 2: Generar Excel con Streams + HTTP
```
node tarea2.js
```
